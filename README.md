This is a working progress for the website. Here is how you can test this for now:

    - download the files on a zip
    - unzip it
    - in the file explorer, go into the foler
    - right click and push "open with terminal"
    - enter "npm run dev"
    - hold shift and left click on the local link

To test mobile:

    - do the above steps, and have open on browser
    - right click and push "inspect"
    - top right of inspect button should be a phone/tablet icon 
    - should be able to adjust it
